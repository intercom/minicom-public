Customer Website

Welcome!

This is a prototype of a customer's website. Your task is to get two way messaging working. You can start by signing up in the upper left hand corner! From there check out the readme.

Structure

/customer-website is the page you are looking at. /admin-website is what the admin would see.

Your framework will have a folder with it's name and have 4 endpoints:

  • POST /customer_api/ping (registers a customer and returns their unread messages)
  • POST /customer_api/read (marks a message as read)
  • GET /admin_api/users (lists all users)
  • POST /admin_api/messages (creates a message for a user)

Feel free to make any changes you need to the server or the front end!

Jquery

To communicate with the Rails app from this website, you'll want to use Jquery. Check out the Jquery documentation for any tips and guidance you might find yourself needing along the way.

Bootstrap

This website is styled using Bootstrap. It's a lightweight way to apply styles that should take some heavy lifting off your plate. For any tips on how to leverage Bootstrap, take a look at the documention.
